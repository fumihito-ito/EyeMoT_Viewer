# EyeMoT Viewer

視線追跡データ（Gaze Data）と動画を同期して可視化・分析するWebビューアです。

## 機能

- **視線可視化** — 動画上に視線軌跡（過去約1秒分）と現在位置をリアルタイム描画
- **時系列グラフ** — Gaze_X / Gaze_Y の時系列チャート表示（Chart.js）
- **マーカー・アノテーション** — 任意の時刻にマーカーを追加し、コメントを付与
- **オフセット調整** — 動画とCSVデータの時間ズレを補正（±0.1 / ±1.0秒）
- **映像フィルター** — 明るさ・コントラスト・彩度・グレースケール等を調整
- **CSVエクスポート** — オフセット・マーカー・コメントをメタデータとして埋め込み保存

## 使い方

1. `index.html` をブラウザで開く
2. 動画ファイル（.mp4 など）と視線データCSVファイルを選択
3. 再生しながら視線データを確認・編集
4. 必要に応じてマーカーを追加し、CSVとして保存

## CSVフォーマット

| 列名 | 内容 |
|------|------|
| Gaze_X | 視線X座標 |
| Gaze_Y | 視線Y座標 |
| Timestamp | タイムスタンプ（`YYYY-MM-DD HH:mm:ss.fff`） |

先頭6行はメタデータ行（日付、スコア、解像度など）として扱われます。  
`# APP_DATA` で始まる行に編集情報（オフセット・マーカー）がJSON形式で保存されます。
